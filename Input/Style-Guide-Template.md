## Usage Notes

When writing new prose using this guide:

1. **Sentence construction:** Build sentences to the type ratios in §1.2, varying length within the range and deviation in §1.1. Open sentences per the distribution in §1.3.
2. **Word selection:** Run candidate words through the five-step filter in §3.5 before committing.
3. **Paragraph shaping:** Target the length range in §2.1 and follow the internal rhythm pattern in §2.2.
4. **Figurative language:** Insert figures at the density in §5.1, drawing from the source domains in §5.3, using the construction patterns in §5.4.
5. **Punctuation:** Match the frequency and usage patterns in §6 — punctuation is one of the most immediately recognizable style markers.
6. **Rhythm check:** Read sentences aloud. Verify pacing matches §7.2 and stress tendency matches §7.1.
